### Concepts Covered:
- **For Loops**
  - Used to execute a block of code a **specific number of times**.
  - Syntax:
    ```java
    for(initialization; condition; update) {
        // code block
    }
    ```
  - Commonly used when the number of iterations is *known in advance*.

- **Loop Control Variable**
  - Acts as the counter for iteration.
  - Must be initialized before the loop starts, checked in the condition, and updated after each iteration.

- **Increment & Decrement**
  - `i++` â†’ increases the loop variable.
  - `i--` â†’ decreases it.
  - You can also use custom steps like `i += 2` or `i -= 3`.

- **Practical Uses**
  - Counting up or down.
  - Running time-based sequences.
  - Repeating actions for a fixed number of times.
