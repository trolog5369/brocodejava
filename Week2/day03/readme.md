### Concepts Covered:
- **While Loops**
  - Executes code repeatedly while a condition is true.
  - Used to validate user input or run tasks until a condition fails.
  - Example: Asking the user for their name or valid age.

- **Do-While Loops**
  - Similar to `while`, but executes the body **at least once**, even if the condition is false.
  - Useful when validation or user input must run once before condition check.

- **Logical Operators**
  - `&&` (AND) â†’ All conditions must be true.
  - `||` (OR) â†’ At least one condition must be true.
  - `!` (NOT) â†’ Inverts the boolean value.
