## ğŸ§  Key Takeaways
- Arrays in Java are **fixed-size** and store elements of the **same data type**.
- Accessing invalid indices throws a runtime exception.
- You can create empty arrays with a defined size, but not modify the size later.
- `Scanner` input handling requires careful buffer management.
- Enhanced `for` loops are clean and efficient for array traversal.
- The `Arrays` class provides useful utilities for sorting and filling arrays.
